## LiguのWave·波动方程·水波物理引擎·v1.0
* 任意水波仿真。
* 通过设置边界条件，可以模拟[干涉],[衍射]等物理教学情境。
* 软件功能：
    [1]. 暂停
    [2]. 初始波面
    [3]. 设置边界条件
        [3.1] 格式:[坐标X, 坐标Y, 幅度A]
        [3.2] 当幅度设"0"，相当于墙，可以模拟衍射
